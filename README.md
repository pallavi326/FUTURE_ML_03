# AI Resume Screening System

## Project Overview
This project is an AI-powered Resume Screening and Ranking System using NLP and Machine Learning.

## Features
- Resume text extraction
- Skill extraction
- Resume-job matching
- Candidate ranking
- Skill gap identification

## Technologies Used
- Python
- NLP
- Scikit-learn
- spaCy
- NLTK

## Output
The system ranks resumes based on similarity with the job description.
